# iCloud 备忘录导入目录

这个目录是从 Apple 备忘录（Notes.app）同步过来的原始笔记文件夹，专门用于博客整理。

## 用途

每次更新后，把 `agent训练/` 等子目录里的笔记内容整理成 `src/content/posts/` 下的博客文章。整理完成后可以删除此目录。

## 目录结构

| 目录 | 内容 |
|------|------|
| `agent训练/` | 实习期间的 Agent 学习笔记、LeetCode 题解、项目分析 |
| `Notes/` | Apple 备忘录导出的其他笔记 |
| `学习篇/` | 学习方法与经验 |

## 整理规则

- 笔记 → 博客文章时，对应关系写在文章的注释或 git log 里
- 图片放在对应文章目录下，命名为 `cover.jpg`（封面）或描述性名称
- 整理完一批后删除此目录，下次更新再重新导入
